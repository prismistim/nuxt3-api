# nuxt3-api-demo

![にほんご]('./README_ja.md')

## 概要
- フロントエンド環境説明用のリポジトリです
- ユーザー情報を扱う簡単なAPIを付属しています
  - DBは用意していないため、データの保存などはできません

## 環境
### フロントエンド
- nuxt3
- nuxtlabs-ui
- vueuse/nuxt
- (typescipt)

### バックエンド
- express
- typescript

## コマンド
### APIサーバーを起動する
```sh
yarn dev:api
```

### フロントエンドのローカルサーバーを起動する
```sh
yarn dev:nuxt3
```
